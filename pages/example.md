# 블로그를 시작하며

안녕하세요! 이 블로그는 **GitHub Pages**와 **Vanilla JavaScript**로 만들어진 정적 블로그입니다.

## 주요 기능

이 블로그는 다음과 같은 기능을 제공합니다:

- 마크다운 기반 게시글 작성
- 다크/라이트 모드 전환
- 태그 기반 필터링
- 클라이언트 사이드 검색
- Giscus 댓글 시스템

## 코드 하이라이팅

코드 블록은 Prism.js를 사용하여 구문 강조가 적용됩니다:

```javascript
// 간단한 JavaScript 예제
function greet(name) {
  console.log(`안녕하세요, ${name}님!`);
}

greet('방문자');
```

Python 코드도 지원됩니다:

```python
# Python 예제
def fibonacci(n):
    if n <= 1:
        return n
    return fibonacci(n - 1) + fibonacci(n - 2)

for i in range(10):
    print(fibonacci(i), end=' ')
```
